Bao gồm tên project
Thư mục chứa controller (code để xử lý các yêu cầu gửi từ view về)
Models chứa các lớp đại diện cho CSDL của ứng dung
Views chứa các thành phần hiển thị giao diện của người dùng
wwwroots thư mục chứa các file của dự án như html css và js
appsettings.json & program.cs chứa các code cấu hình của hệ thống
